# PoliStrat 학생용 게임 전략 설명서

이 게임에서 좋은 전략은 “가장 센 행동을 고르는 것”이 아니라 “우리 국가의 약점과 이번 턴 상황에 맞는 선택을 하는 것”이다. 매 턴 자원, 외교, 지도, 순위를 보고 한 가지 목표를 정하자.

## 1. 먼저 우리 국가를 읽어라

| 우리 상태 | 추천 방향 |
|---|---|
| 예산이 많다 | 산업 투자, 군비 증강, 식량 확보, 에너지 개발을 적극 활용 |
| 군사력이 높다 | 영토 점령이나 견제 전략 가능 |
| 외교가 높다 | 무역 협정, 정상회담, 에너지 수입 협정으로 성장 |
| 식량/에너지가 낮다 | 위기 이벤트에 취약하므로 안정화 우선 |
| 지지율이 낮다 | 방어 태세, 정상회담, 인도적 지원으로 회복 |
| 핵심 점수가 낮다 | 핵심 지역 점령이나 공동 제재 검토 |

## 2. 초반, 중반, 후반 전략

| 시기 | 핵심 목표 | 좋은 선택 |
|---|---|---|
| 1턴 | 국가 상태 파악, 위험 줄이기 | 전략회의에서 약점 확인, 외교 관계 만들기 |
| 2~3턴 | 성장 방향 확정 | 경제 성장, 외교 협정, 군사력 보강 |
| 4~5턴 | 점수 마무리 | 핵심 지역 확보, 타이틀 조건 달성, 약점 보완 |

초반부터 무리하게 군사 행동만 하면 자원이 빨리 마른다. 반대로 너무 안전하게만 가면 핵심 점수와 순위 경쟁에서 밀릴 수 있다.

## 3. 행동 선택 전략

| 행동 | 언제 좋은가 | 주의할 점 |
|---|---|---|
| 영토 점령 | 군사력이 충분하고 지도 확장이 필요할 때 | 핵심 지역은 비용이 크다. 군사력 부족 주의 |
| 군비 증강 | 다음 턴 점령이나 방어가 필요할 때 | 예산을 계속 쓰면 경제 행동이 어려워진다 |
| 방어 태세 | 공격을 받을 것 같거나 지지율도 올리고 싶을 때 | 직접 점수 상승은 크지 않다 |
| 산업 투자 | GDP와 최종 점수를 키우고 싶을 때 | 지지율 -1을 감수해야 한다 |
| 식량 확보 | 식량이 낮거나 위기 대응이 필요할 때 | 당장 순위 상승은 작을 수 있다 |
| 에너지 개발 | 에너지가 낮거나 장기 성장이 필요할 때 | 지지율 -1을 감수해야 한다 |

## 4. 외교 전략

외교는 “친하게 지내자”가 아니라 조건을 주고받는 과정이다. 제안할 때는 다음 세 가지를 분명히 하자.

1. 우리가 원하는 것
2. 상대가 얻는 것
3. 서로 지킬 약속

| 협정 | 좋은 상황 |
|---|---|
| 동맹 선언 | 서로 군사 충돌을 피하고 신뢰를 만들 때 |
| 무역 협정 | 양쪽 모두 경제 성장이 필요할 때 |
| 에너지 협정 | 한쪽은 에너지, 다른 쪽은 예산이나 외교 이익이 필요할 때 |
| 분쟁 중재 | 외교 타이틀이나 전략 점수를 노릴 때 |
| 공동 제재 | 특정 강국을 견제하거나 핵심 점수가 필요할 때 |

## 5. 타국을 읽는 법

지도에서 다른 국가 타일을 누르면 상세 현황을 볼 수 있다. 다음 질문으로 판단하자.

- 저 국가는 군사력이 높은가?
- 저 국가는 식량이나 에너지가 부족한가?
- 저 국가는 우리와 협력하면 이익이 있는가?
- 저 국가가 다음 턴에 우리를 견제할 가능성이 있는가?
- 지금 1위 국가는 어떤 자원이 강한가?

강한 국가는 무조건 적이 아니다. 1위와 협력해 안전을 얻을 수도 있고, 여러 국가가 함께 견제할 수도 있다.

## 6. 승리 타이틀을 노려라

최종 점수 1위만이 목표는 아니다. 우리 국가의 특징에 맞는 타이틀을 노리면 발표에서 전략을 설명하기 쉽다.

| 타이틀 | 노릴 방향 |
|---|---|
| 종합 패권국 | 전체 점수 1위 |
| 군사 패권국 | 군사력 최고 |
| 외교 중재국 | 외교 최고 |
| 경제 강국 | GDP 최고 |
| 핵심 거점 장악국 | 핵심 점수 최고 |
| 생존 안정국 | 식량 80 이상, 에너지 70 이상 |
| 국내 결속 우수국 | 지지율 75 이상 |

## 7. 마지막 조언

- 한 턴에 목표를 너무 많이 잡지 말자.
- 제출 전 “비용을 낼 수 있는가?”를 확인하자.
- 말로 한 외교 약속은 활동지에 적자.
- 1위만 보지 말고, 우리보다 바로 앞선 국가를 비교하자.
- 발표할 수 없는 선택은 좋은 전략이 아니다. 선택 이유를 한 문장으로 말할 수 있어야 한다.
