# PoliStrat 학생용 게임 규칙 설명서

PoliStrat은 국제정치 상황 속에서 국가의 자원, 외교, 군사력, 국내 지지율을 관리하며 5턴 동안 가장 강한 국가를 만드는 전략 시뮬레이션 게임이다. 각 모둠은 하나의 국가를 맡고, 태블릿 1대로 토론한 뒤 공동 결정을 제출한다.

## 1. 게임 목표

최종 점수가 높은 국가가 우세하다. 단순히 군사력만 높다고 이기는 것이 아니라 경제, 외교, 국내 안정, 핵심 거점 확보가 함께 중요하다.

점수는 다음 요소를 바탕으로 계산된다.

| 요소 | 의미 |
|---|---|
| GDP | 국가의 경제 규모 |
| 군사력 | 영토 점령과 안보 능력 |
| 외교 | 협상, 협정, 국제 영향력 |
| 지지율 | 국내 정치 안정 |
| 방어 태세 | 공격과 위기에 대비한 준비 |
| 핵심 점수 | 전략적으로 중요한 지역 확보 |

게임 종료 후에는 점수와 특징에 따라 `종합 패권국`, `군사 패권국`, `외교 중재국`, `경제 강국`, `핵심 거점 장악국`, `생존 안정국` 등의 타이틀을 받을 수 있다.

## 2. 한 턴의 흐름

한 게임은 최대 5턴으로 진행된다. 각 턴은 네 단계로 나뉜다.

| 단계 | 시간 | 할 일 |
|---|---:|---|
| 전략회의 | 3분 | 현재 자원, 지도, 순위, 이벤트를 확인하고 이번 턴 목표를 정한다. |
| 외교 페이즈 | 3분 | 다른 국가와 협상하고 협정이나 공동 대응을 제안한다. |
| 행동 입력 | 2분 | 모둠의 최종 행동을 하나 선택해 제출한다. |
| 결과 처리 | 1분 | 교사가 결과를 반영하고 각 국가의 변화를 확인한다. |

## 3. 자원 읽는 법

| 자원 | 쓰임 |
|---|---|
| 예산 | 군비 증강, 산업 투자, 식량 확보, 에너지 개발 등에 사용 |
| 군사력 | 영토 점령과 안보 전략에 사용 |
| 외교 포인트 | 외교 행동과 협상에 사용 |
| GDP | 최종 점수와 국가 성장의 핵심 |
| 지지율 | 국내 안정과 최종 평가에 영향 |
| 식량 | 위기 대응과 생존 안정에 중요 |
| 에너지 | 경제와 위기 대응에 중요 |

자원은 높을수록 좋지만, 대부분의 행동은 어떤 자원을 소모하고 다른 자원을 얻는 방식이다. 무조건 많이 쓰기보다 다음 턴까지 버틸 수 있는 균형이 필요하다.

## 4. 외교 페이즈 행동

| 행동 | 비용 | 효과 |
|---|---|---|
| 에너지 수입 협정 | 외교 -8 | 에너지 +12 |
| 정상회담 | 외교 -5 | 외교 +6, 지지율 +3 |
| 무역 협정 | 외교 -6 | GDP +8, 예산 +4, 지지율 +1 |
| 공동 제재 | 외교 -10 | 핵심 +2, 지지율 +2 |
| 인도적 지원 | 예산 -8 | 외교 +4, 지지율 +6 |

외교 제안 메뉴에서는 동맹 선언, 무역 협정, 에너지 협정, 분쟁 중재, 공동 제재를 제안할 수 있다. 제안은 교사가 승인하거나 거절한다.

## 5. 행동 입력 단계 행동

| 행동 | 비용 | 효과 |
|---|---|---|
| 영토 점령 | 일반 군사 -20 / 핵심 군사 -30 | 점령 +1, 핵심 지역이면 핵심 +3 |
| 군비 증강 | 예산 -10 | 군사력 +8 |
| 방어 태세 | 예산 -8 | 군사력 +4, 지지율 +2, 방어 +1 |
| 산업 투자 | 예산 -12 | GDP +12, 지지율 -1 |
| 식량 확보 | 예산 -8 | 식량 +12, 지지율 +1 |
| 에너지 개발 | 예산 -10 | 에너지 +10, GDP +3, 지지율 -1 |

영토 점령은 반드시 지도에서 타일을 먼저 선택해야 한다. 별표 핵심 지역은 비용이 크지만 보상도 크다.

## 6. 중요한 규칙

- 한 턴에 모둠 행동은 하나만 제출한다.
- 제출 후에는 교사가 취소하지 않는 이상 바꿀 수 없다.
- 자원이 부족하면 행동을 제출할 수 없다.
- 외교 약속은 말로만 끝나지 않게 조건을 분명히 적는다.
- 다른 국가의 상세 현황은 지도에서 해당 국가 타일을 눌러 확인한다.
- 게임의 핵심은 승리만이 아니라, 왜 그런 선택을 했는지 설명하는 것이다.
