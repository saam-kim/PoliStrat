# PoliStrat 교사용 진행 매뉴얼

PoliStrat은 모둠별 국가 운영을 통해 국제정치의 핵심 요소인 안보, 경제, 외교, 국내 정치, 자원 안정을 경험하게 하는 전략 시뮬레이션 게임이다. 교사는 전자칠판 또는 교사용 컴퓨터에서 대시보드를 열고, 학생은 모둠당 태블릿 1대로 참가한다.

## 1. 수업 전 준비

1. 교사용 화면에서 Firebase 설정이 저장되어 있는지 확인한다.
2. 교사용 PIN `1234`를 입력하고 교사 화면을 시작한다.
3. 생성된 6자리 세션 코드를 학생들에게 안내한다.
4. 학생들은 국가/모둠 이름을 입력하고 참가한다.
5. 모든 모둠이 들어오면 `참여 모둠 확정`을 누른다.
6. 가능하면 `리허설 데이터 생성`으로 한 번 흐름을 점검한 뒤 실제 세션을 시작한다.

## 2. 한 턴 운영 흐름

| 단계 | 시간 | 교사 역할 |
|---|---:|---|
| 전략회의 | 3분 | 이벤트나 공지를 안내하고, 학생들이 자원·지도·순위를 보게 한다. |
| 외교 페이즈 | 3분 | 협상 시간을 주고, 협정 현황에서 제안을 승인/거절한다. |
| 행동 입력 | 2분 | 제출 현황을 보며 미제출 모둠을 안내한다. |
| 결과 처리 | 1분 | 다음 페이즈 이동 시 자동 반영된 결과와 턴 요약을 확인한다. |

페이즈 이동은 상단의 `다음` 버튼으로 진행한다. 잘못 넘겼다면 `이전` 버튼을 사용할 수 있다.

## 3. 교사용 대시보드 읽기

| 영역 | 활용 방법 |
|---|---|
| 참가 국가 | 국가별 자원, 최근 행동, 점령·핵심 상황 확인 |
| 지도 | 국가별 영토와 핵심 지역 확인 |
| 제출 현황 | 현재 턴 제출 여부 확인 |
| 협정 현황 | 외교 제안 승인/거절 |
| 턴 요약 | 결과 처리 후 변화 확인 |
| 실시간 이벤트 로그 | 제출, 협정, 처리 기록 확인 |
| 국력 순위 | 현재 점수 흐름 확인 |

## 4. 운영 도구

| 도구 | 사용 상황 |
|---|---|
| 공지 | 전체 안내나 시간 압박을 로그에 남길 때 |
| 이벤트 | 이번 턴 국제 정세 변수를 줄 때 |
| 마감/재개 | 제출을 잠그거나 다시 열 때 |
| 선택 국가 자원 반영 | 교사가 수동으로 보정할 때 |
| 타일 소유권 변경 | 지도 오류나 수업상 조정이 필요할 때 |
| 선택 국가 제출 취소 | 학생이 실수로 제출했을 때 |
| 이번 턴 처리 되돌리기 | 결과 처리 후 오류를 발견했을 때 |
| 백업/복원 | 수업 중 데이터 보호가 필요할 때 |
| 발표 | 최종 순위와 타이틀을 보여줄 때 |

## 5. 수업 진행 팁

- 첫 턴은 규칙 적응용으로 운영한다. 점수 경쟁보다 화면 읽는 법과 제출 흐름을 익히게 한다.
- 외교 페이즈에서는 “누가 무엇을 얻는가”를 반드시 말하게 한다.
- 행동 입력 전 30초에는 “최종 행동 하나만 제출”을 다시 안내한다.
- 영토 점령은 타일 선택이 필요하므로, 제출 전 지도 선택 여부를 확인하게 한다.
- 결과 처리 후에는 순위보다 “왜 이런 변화가 생겼는가”를 먼저 묻는다.

## 6. 예상 문제와 대응

| 문제 | 대응 |
|---|---|
| 학생이 교사용 화면에 들어가려 함 | 교사용 PIN을 알려주지 않는다. |
| 학생이 잘못 제출함 | 해당 국가 선택 후 `선택 국가 제출 취소` |
| 교사가 페이즈를 잘못 넘김 | `이전` 버튼 사용 |
| 결과 처리 후 오류 발견 | `이번 턴 처리 되돌리기` 사용 |
| 화면을 새로고침해야 함 | 이전 게임 기록 또는 세션 코드로 복구 |
| 수업 시간이 부족함 | 3턴까지만 진행하고 발표 화면으로 마무리 |

## 7. 마무리 질문

- 가장 강한 국가는 어떤 전략을 사용했는가?
- 외교 협력은 실제로 이익이 되었는가?
- 군사력, 경제력, 외교력 중 무엇이 가장 중요했는가?
- 우리 모둠의 선택 중 다시 한다면 바꾸고 싶은 것은 무엇인가?
