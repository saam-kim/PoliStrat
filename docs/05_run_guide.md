# PoliStrat 실행 안내

## 1. 교사용 컴퓨터에서 실행

1. 프로젝트 폴더에서 `start_server.bat`을 실행한다.
2. 터미널 창이 열리면 닫지 않는다.
3. 브라우저에서 `http://127.0.0.1:8000/index.html`을 연다.
4. 교사용 PIN `1234`를 입력하고 교사 화면을 시작한다.
5. 화면에 표시된 6자리 세션 코드를 학생들에게 안내한다.

## 2. 학생 접속

학생 태블릿이 같은 네트워크에서 접속해야 한다. 교사용 컴퓨터의 IP 주소를 확인한 뒤 학생에게 아래 주소 형식으로 안내한다.

`http://교사용컴퓨터IP:8000/index.html`

예시:

`http://192.168.0.23:8000/index.html`

학생은 교사가 알려준 6자리 세션 코드와 국가/모둠 이름을 입력해 참가한다.

## 3. 수업 전 점검

교사용 대시보드의 `운영 도구`에서 `수업 전 점검`을 누른다. 다음 항목을 확인한다.

- Firebase 연결
- 교사 세션 생성 또는 복구
- 참가 국가 2개 이상
- 참여 모둠 확정
- 현재 페이즈 확인
- 브라우저 백업 저장 가능 여부

## 4. 수업 중 문제 해결

| 상황 | 해결 |
|---|---|
| 학생이 잘못 제출함 | 해당 국가를 선택하고 `선택 국가 제출 취소` |
| 교사가 페이즈를 잘못 넘김 | 상단 `이전` 버튼 |
| 결과 처리 후 오류 발견 | `이번 턴 처리 되돌리기` |
| 브라우저 새로고침 필요 | 이전 게임 기록 또는 세션 코드로 복구 |
| 수업을 마쳐야 함 | `발표` 버튼으로 결과 발표 화면 열기 |

## 5. 결과 저장

최종 발표 화면에서 `결과 파일 저장`을 누르면 순위와 국가별 주요 수치가 `.tsv` 파일로 저장된다. 이 파일은 Excel, Google Sheets, 한글 표, Word 표에 붙여 넣어 정리할 수 있다.
